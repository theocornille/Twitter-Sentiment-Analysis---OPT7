Projet OPT7

Théo CORNILLE - Luc GIBAUD - Cédric DES LAURIERS - Mirwaisse DJAMBAZ

Twitter Sentiment Analysis

2 datasets:
dataset d'1.6 millions de tweets : à télécharger ici https://www.kaggle.com/kazanova/sentiment140
dataset de 15 000 tweets de clients de compagnies aériennes américaines

4 différentes parties dans ce projet:
- Analyse de sentiments à partir d'un dataset d'1.6 millions de tweets
- Transfer Learning sur un dataset de 15 000 tweets de clients de compagnies aériennes américaines.
- Topic modeling sur ce deuxième dataset pour classifier les différents types de problèmes des tweets négatifs.
- Analyse de tweets en temps réel via l'API de twitter : Analyse de sentiments + Topic prediction
